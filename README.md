# TicTacToe Game 🎮

A classic Tic Tac Toe game built using Java Swing with a graphical user interface.

## About
Two players can play this game on the same computer.
Player 1 plays as **O** (Yellow) and Player 2 plays as **X** (Red).
The game automatically detects wins and draws.

## Features
- Interactive GUI built with Java Swing
- Two player gameplay
- Win detection for all 8 possible combinations
- Draw detection
- Color coded players (Yellow for O, Red for X)
- Restart or exit option after each game

## Technologies Used
- Java (Core)
- Java Swing (GUI)
- AWT (Layout and Events)

## How to Run
1. Clone the repository
   git clone https://github.com/MahajanPyanshu/TicTacToe.git
2. Open project in IntelliJ IDEA
3. Run Main.java

## Gameplay
- Player 1 → O (Yellow)
- Player 2 → X (Red)
- Players take turns clicking buttons
- First to get 3 in a row wins
- If all 9 boxes filled with no winner → Draw
